[EN]
My bot code is open source. You can create a similar bot locally if you have a friend who has Telegram Premium.
This bot allows you to use premium emojis EVEN if you DON'T have Premium!
To do this, use the %Emoji_ID% placeholder, where emoji_id is the emoji ID.
Currently, all formatting types are supported: photos, files, GIFs, plain text, and videos, regardless of size and length.
